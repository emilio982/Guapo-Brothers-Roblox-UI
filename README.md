# Guapo Brothers Roblox UI

A modern beige-themed Roblox client-side utility UI.

## Features
- Password system
- Player tab (Speed, Fly, Noclip)
- Combat tab (NPC Aimbot - camera)
- Discord button
- Studio-safe LocalScript

## Installation
1. Open Roblox Studio
2. Place a LocalScript in:
   StarterPlayer > StarterPlayerScripts
3. Paste the script inside
